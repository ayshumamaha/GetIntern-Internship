# Terminal-Based Inventory / Task Management Engine

## Project Overview

This project is a menu-driven command-line application that manages inventory or task records using a persistent JSON database. Users can add, view, update, and delete records while ensuring data is saved between program executions.

## Features

* Interactive menu-driven interface
* Add new records
* View all records
* Update existing records
* Delete records
* Persistent JSON storage
* Input validation
* Exception handling

## Technologies Used

* Python
* json (built-in library)
* os
* datetime

## Files

* `inventory_manager.py` – Main Python program
* `database.json` – Stores all records

## Run

```bash
python inventory_manager.py
```
